# Splunk SIEM Security Analytics Projects

Hey folks!\
This repository contains **Splunk SIEM dashboards** that analyze security logs, detect anomalies, and monitor network/email traffic.

---

## Projects Included

| Project | Category | Description |
|---------|----------|-------------|
| **SMTP Log Analysis Dashboard** | Email Security | Monitors SMTP traffic to detect failures, spam attempts, and suspicious IP behavior |
| **Cross-Layer Network Security Analytics Dashboard** | SOC / Threat Detection | Correlates OS, Network, System and Server logs to detect multi-stage cyber attacks |
