# SMTP Log Analysis

A Splunk SIEM dashboard to visualize SMTP email traffic, detect failures, and identify suspicious source IPs.

---

## Features

- Top **HELO/EHLO** domain attempts
- SMTP success vs failure visualization
- SMTP response codes (`221`, `250`, `535`, etc.)
- Top source & destination IPs

---

## Industry Use Case

Email security teams use SMTP log dashboards to:
- Detect spam or brute-force SMTP attempts
- Identify SMTP relay abuse / spoofing
- Track mail delivery performance & failure spikes
