# Cross-Layer Network Security Analytics

A Splunk dashboard that correlates logs from **Network, OS, System and Server layers** to detect anomalies and multi-stage cyber attacks.

---

## Features
- Deny/Allow Traffic Trend Over Time
- Unified Event Timeline (Network + OS + Server + System)
- Top Blocked Source and Destination IPs
- Log Event Ratio (Errors/Warnings/Fail/Success)
- Failed Login Trend Analysis (Brute-force Detection)

---

## Why This Matters (Industry Use Case)

SOC teams use cross-layer correlation to:
- Detect lateral movement inside the network
- Identify unauthorized access attempts
- Monitor firewall denies and malicious network activities
